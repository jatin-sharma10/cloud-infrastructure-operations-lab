Project Overview

This project demonstrates a production-style cloud infrastructure deployment built entirely on AWS Free Tier. The environment showcases Linux system administration, Docker containerization, web server deployment, AWS IAM, CloudWatch monitoring, centralized logging, and infrastructure operations following cloud engineering best practices.
